# logistic-map-demo
# Визуализация логистического отображения (теория хаоса) на PyQt6

![Интерфейс приложения](screenshot.png)  
*Пример работы: бифуркационная диаграмма и итерации для r = 3.6*

## О проекте

Интерактивное приложение для исследования **логистического отображения** — классической модели, демонстрирующей переход к хаосу через удвоение периода. С помощью слайдера можно менять параметр роста `r` и в реальном времени наблюдать:

- Временной ряд последних итераций;
- Накопление бифуркационной диаграммы;
- Вычисление показателя Ляпунова (критерий хаоса).

Проект создан для портфолио и демонстрирует навыки разработки GUI на PyQt6, работы с численными методами и интерактивной визуализации данных.

## Возможности

- 🎛️ **Управление параметром `r`** от 2.5 до 4.0 с помощью слайдера.
- 📈 **График итераций** — отображение последних 200 шагов.
- 🌀 **Бифуркационная диаграмма** — накапливается по мере изменения `r`.
- 📊 **Показатель Ляпунова** — рассчитывается автоматически для текущего `r`.
- ⚡ **Быстрая отрисовка** благодаря `pyqtgraph`.

## Технологии

- Python 3.13+
- PyQt6 — графический интерфейс
- PyQtGraph — интерактивные графики
- NumPy — численные расчёты
